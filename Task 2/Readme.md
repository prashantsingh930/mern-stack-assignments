# Laundry Wallah / Laundry Mart

A simple HTML webpage for a laundry service business. This project demonstrates basic HTML concepts such as headings, paragraphs, images, lists, tables, forms, and footers.

## Features

* Business introduction section
* Laundry service listing
* Price table for services
* Customer booking form
* Contact information footer
* Clean and beginner-friendly HTML structure

## Technologies Used

* HTML5

## Project Structure

```
project-folder/
│
├── index.html
├── laundary.avif
└── README.md
```

## Services Offered

* Washing and Drying
* Ironing and Folding
* Dry Cleaning
* Stain Removal
* Express Service
* Special Garment Care

## How to Run

1. Download or clone the project.
2. Ensure `laundary.avif` is in the same folder as `index.html`.
3. Open `index.html` in any web browser.
